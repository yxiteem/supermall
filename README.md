# supermall
描述：一个类似于蘑菇街的移动端商城的项目，具有首页，分类页，购物车页，我的四个tabbar页面，采用采用组件化思维、面向对象思想构建整个项目，采用axios请求数据，通过组件间传值实现数据的展示。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
